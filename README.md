# MOOCChain Web

Frontend web application for MOOCChain blockchain MOOC platform, built with React, TypeScript, and Vite.

## Tech Stack

- **Framework**: React 19.x
- **Language**: TypeScript 5.9
- **Build Tool**: Vite 7.x
- **Linting**: ESLint 9.x with TypeScript support

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

App runs on `http://localhost:5173` with hot module replacement (HMR).

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
